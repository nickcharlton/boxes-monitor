# boxes-monitor

A utility for monitoring the dependencies [boxes][] depends on and reporting
them.

## Monitored Dependencies

* ISO updates
    - [Ubuntu][]
    - [Debian][]
* [VirtualBox versions][]
* GitHub projects.
    - [chruby][]
    - [ruby-install][]
